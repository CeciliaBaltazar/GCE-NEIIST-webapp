Esta secção explica os procedimentos que tens de tomar para te inscreveres no Hub do GCE. Lê todos os passos com muita atenção! (clica em cada passo para veres mais). Além disso, tentamos esclarecer algumas dúvidas adicionais que possas ter.

Passo 1 - Arranjar equipa Hash Code

As equipas são de pelo menos 2 elementos. Se ainda não tiveres equipa, podes postar no nosso evento.
Passo 2 - Pré-Inscrição Hash Code

Se ainda não o fizeste, por favor pré-inscreve a tua equipa. A pré-inscrição permite-nos apurar quantas pessoas vão querer participar, de modo a que possamos alocar recursos para todos. Nota que a pré-inscrição só garante o lugar da tua equipa na lista de espera.
Passo 3 - Inscrição no Judge System da Google Hash Code

Cada elemento da equipa tem de se inscrever aqui. Nota que é importante que todos o façam a tempo, pois a criação da conta pode levar até 24 horas. Depois da inscrição, há que formar equipa no Judge System (ver FAQ da Google).

Depois de seguires estes passos todos, estás quase lá. Só falta juntares-te ao Hub do GCE (ver passo 4).

Passo 4 - Junta-te ao Hub do GCE Hash Code

Finalmente, e para garantires a tua presença dia 1 de março no nosso evento, tu e os teus colegas de equipa têm de se juntar ao nosso Hub. Se reparares, de momento o nosso Hub não está a aceitar inscrições. As inscrições irão abrir dia 1 de fevereiro às 14:30h, e irão encerrar no mesmo dia às 16:55h. Será first come, first served. Por isso, garante que às 14:30h todos os membros estão prontos para se juntar ao Hub. Nota que quando alcançarmos o limite de capacidade as inscrições irão encerrar.
