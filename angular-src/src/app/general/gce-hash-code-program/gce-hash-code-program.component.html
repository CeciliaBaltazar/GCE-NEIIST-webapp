Google Hash-Code
Procedimentos

O evento decorrerá no dia 1 de março de 2018, quinta-feira, das 14:30h às 22h, no Pavilhão de Informática I, Instituto Superior Técnico, (local a designar). O GCE organizará uma sessão de apresentação do Google Hash Code, explicação das regras e esclarecimento de dúvidas. Haverá também uma pequena sessão de talks, dada pelos nossos parceiros. Seguidamente, os participantes serão encaminhados para o coffee-break, onde poderão conhecer os nossos parceiros e as suas ofertas. Depois do coffee-break, a competição terá início.

Programa

Atividade Horário
Check-in 14:30h-15:00h

Sessão de Networking (com coffee-break!) 16:00h-17:00h
Competição 17:00-21:30h
Resultados 21:30h-22:00h

As pré-inscrições/waiting-list decorrerão a partir de 24 de janeiro. Dia 1 de fevereiro as inscrições para o hub do GCE estarão oficialmente abertas. Embora a pré-inscrição não seja obrigatória para se participar no Google Hash Code, esta permite-nos estimar o número de participantes e, consequentemente, alocar recursos suficientes para todos. A pré-inscrição não garante lugar na competição, mas garante lugar na lista de espera.
