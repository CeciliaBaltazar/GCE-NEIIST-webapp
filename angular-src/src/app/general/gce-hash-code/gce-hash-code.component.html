O evento decorrerá no dia 1 de março de 2018, quinta-feira. O GCE organizará uma sessão de apresentação do programa, explicação das regras e esclarecimento de dúvidas, assim como uma pequena sessão de talks. Seguidamente, os participantes serão encaminhados para a sala de competição.

As pré-inscrições decorrerão de 22 de janeiro a 28 de janeiro. Dia 28 de janeiro as inscrições para o hub do GCE estarão oficialmente abertas. Embora a pré-inscrição não seja obrigatória, esta permite-nos estimar o número de participantes e, consequentemente, alocar recursos suficientes para todos. A pré-inscrição não garante lugar na competição, apenas serve como lista de espera.
